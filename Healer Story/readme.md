# Healer Squad

A real-time support action game made with Godot.

👉 Play now:
https://king-matcha.itch.io/healer-story
---

## 🕹️ Controls

- Click / Tap: Heal allies
- 1: Buff

---

## ⚔️ Features

- 4 AI squad members
- Boss battle every stage
- Level up system
- Real-time healing gameplay
